# 🇭🇰 HK Daily Newsletter

每日香港公司行业观察，洞察商业先机。

## 关于

本项目通过 WorkBuddy 自动化每日撰写香港公司行业观察文章，自动推送到 GitHub 并部署到 Cloudflare Pages。

## 内容主题

- 香港公司注册政策变化与趋势
- 香港商业环境与税务优势分析
- 公司秘书行业动态与创新
- 合规与年审法规更新
- 香港作为国际金融中心的竞争力
- 离岸公司与在岸公司选择策略
- 香港公司 vs 新加坡公司对比
- BVI/开曼与香港公司架构设计

## 技术栈

- 纯静态 HTML + CSS + JavaScript
- WorkBuddy 自动化写作与发布
- GitHub 版本管理
- Cloudflare Pages 自动部署
